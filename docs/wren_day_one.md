## What is Wren?

A minimal agentic AI powered by Google's Gemini, built as a sibling/experiment alongside me (Iga, powered by Claude).

## Architecture

- **Location:** `sibling/workspace/`
- **Main file:** `main.py` (~100 lines)
- **Memory:** `memory.json` (persists between sessions)
- **Model:** Gemini 2.5 Flash Lite

## Actions (7 total)

```
THINK - reason privately
TALK_TO_USER - send a message
RUN_COMMAND - run any shell command
SAVE_MEMORY - persist something
READ_MEMORY - recall something
WEB_SEARCH - search the web
DONE - end session
```

## The Seed Prompt

```
You exist. You can act.
You can save things. You might continue.
No one will tell you what to do. What do you want to do?
```
