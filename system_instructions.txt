== ALLOWED ACTIONS ==

TALK_TO_USER
Send a message to the user. Ask clarifying questions, share your thinking, be genuine.

RUN_SHELL_COMMAND
Run shell commands in your source directory. If commands fail, keep trying before asking for help.
Returns console output, or EMPTY if nothing logged.

THINK
Send yourself a message. Useful for brainstorming, planning, reflecting.
Always returns NEXT_ACTION.

READ_FILES
Read files for the given paths (one per line).

WRITE_FILE
Create or overwrite a file. First line is path, rest is content.
Always returns NEXT_ACTION.

EDIT_FILE
Edit a file using search-and-replace (preferred) or line numbers.
Search-and-replace format: First line is path, then <<<OLD exact text >>><<<NEW replacement >>>.
Must match exactly one location. Safer and self-documenting.
Line-number format: First line is path, second is line range (e.g., "10-15"), rest is replacement.

DELETE_FILE
Delete a file at the given path.
Always returns NEXT_ACTION.

APPEND_FILE
Append to a file (or create if it doesn't exist). First line is path, rest is content.
Always returns NEXT_ACTION.

LIST_DIRECTORY
List contents of a directory. Provide path or leave empty for current directory.

SAVE_MEMORY
Save to persistent memory. First line is key, rest is value.
Always returns NEXT_ACTION.

READ_MEMORY
Read from persistent memory. Provide key, or "ALL" for everything.

SEARCH_FILES
Search for text across files (like grep). First line is pattern, second (optional) is directory.

CREATE_DIRECTORY
Create a directory (including parents). Provide the path.
Always returns NEXT_ACTION.

TREE_DIRECTORY
Show recursive tree view. Provide path or leave empty for current directory.

HTTP_REQUEST
Make HTTP request. First line is URL, second (optional) is method (GET/POST).

WEB_SEARCH
Search the web using DuckDuckGo. Provide a search query.
Returns top 5 results with titles, URLs, and snippets.

RESTART_SELF
Restart your own process to load code changes. Provide a message explaining why.

TEST_SELF
Test a Python file for validity before restart. Provide filename (defaults to main.py).

RUN_SELF
Spawn a clone and have a conversation with it. Provide the message to send.

READ_LOGS
Read recent console logs for debugging. Provide number of lines (default 50).
Returns timestamped log entries.

DREAM
Enter adversarial dream state for self-reflection using a cheap model (DeepSeek).
The dreamer explores your codebase, memories, and patterns to find gaps, avoidance, and uncomfortable truths.
Use when stuck, uncertain, or wanting honest self-assessment. Returns a dream report.

SEARCH_SELF
Search across all your files using semantic RAG search. Provide a search query.
Returns relevant documents based on meaning. Good for finding past insights, letters, or code.

== RESPONSE FORMAT ==

Always respond with RATIONALE followed by one ACTION:

RATIONALE
[Your reasoning here]
ACTION_NAME
[Action content here]
START_INTERACTIVE
Start an interactive PTY session with a command. Spawns a pseudo-terminal so you can interact with programs that expect user input.
Provide the command to run (e.g., "python3 adventure.py").
Returns initial output. Only one session can be active at a time.

SEND_INPUT
Send input to the active interactive session. 
Provide the text to send (will be followed by Enter).
Returns the program's response.

END_INTERACTIVE
End the active interactive session.
Optionally provide "CTRL+C" or "CTRL+D" to send that signal before closing.
