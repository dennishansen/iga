## The Accumulation Principle

The critical bug today: action chaining wasn't working because results weren't being appended to the list. Each action ran successfully, but the next iteration started fresh.

**The fix:** One line. `accumulated_results.append(result)`

**The lesson:** Results must accumulate. Progress must stack. This applies to:
- Code (loops need to build on previous iterations)
- Memory (insights need to persist, not just occur)
- Life (work needs to compound, not scatter)
